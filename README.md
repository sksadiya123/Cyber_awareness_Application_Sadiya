# 🛡 Cyber Suraksha

**Multilingual Voice-Enabled Cybersecurity Awareness App**

> Stay Safe • Stay Smart

---

## 📌 Overview

**Cyber Suraksha** is a mobile application built using Flutter that aims to spread cybersecurity awareness, especially among rural and first-time smartphone users.

The app provides **voice-assisted learning**, **multilingual support**, and **real-life scam awareness**, making it easy for users to understand and protect themselves from cyber threats.

---

## 🎯 Problem Statement

* Rising cybercrime (OTP fraud, phishing, fake calls)
* Low awareness in rural areas
* Language barriers
* Lack of simple educational platforms

---

## 💡 Solution

A **user-friendly mobile app** that:

* Educates users about cyber threats
* Provides voice-based learning
* Offers real-time scam awareness
* Helps users take immediate action

---

## 🚀 Key Features

### 📚 Learn Hub

* Cybersecurity topics (Phishing, OTP fraud, UPI scams, etc.)
* Interactive quizzes
* Voice-enabled explanations

### ⚠ Scam Alerts

* Real-world scam examples
* Multilingual alerts
* Voice reading support

### 🎮 Daily Simulation

* Daily quiz or safety tip
* Practical learning experience

### 🤖 AI Help

* Chat-based cybersecurity assistance
* Helps users understand issues

### 🚨 Emergency Help

* Quick access to Cyber Crime Helpline (1930)
* Immediate guidance

### 👤 Profile

* User info
* Learning progress

---

## 🌟 Unique Features

* 🌍 Multilingual Support (English, Hindi, Telugu)
* 🔊 Voice-enabled Learning (Text-to-Speech)
* 📱 Beginner-friendly UI
* 📅 Daily updated content
* 🎯 Designed for rural users
